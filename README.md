# go_rest_api
Go web server with mongoDb

Dependancies
  - go get gopkg.in/mgo.v2
  - go get -u github.com/gorilla/mux
  - go get -u github.com/gorilla/handlers
  - go get -u golang.org/x/crypto/bcrypt
